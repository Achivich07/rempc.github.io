Поиск вирусов и диагностика Windows

Набор утилит для поиска и удаления вредоносных программ:

ADW Cleaner [ сайт ] Утилита от Malwarebytes – удаляет вирусы, рекламное ПО и другие нежелательные программы.

Rogue Killer [ сайт / видео ] Отличная утилита, очень дотошная, детально удаляет вирусы, рекламное ПО и другие нежелательные программы.

AVZ [ сайт ] Антивирус Зайцева, многопрофильная утилита про нахождению заразы и исследованию системы.

Unlocker [ сайт ] Удаляет файлы и папки, которые заняты другими приложениями.

Чистка, настройка, оптимизация

Набор программ для чистки системы, настройки Windows, отключения лишних служб, ускорения загрузки: 

Wise Disk Cleaner [ сайт ] Утилита для очистки мусора в Windows 7,8,10, включая дистрибутивы обновлений и временные файлы приложений.

Wise Reg Cleaner [ сайт ] Утилита для чистки и оптимизации реестра в Windows 7,8,10.

CCleaner [ сайт ] Утилита для очистки мусора в Windows 7,8,10, в комплекте также идет очистка реестра и мониторинг автозагрузки.

Space Sniffer [ сайт ] Визуально отображает размер папок и файлов на диске. Позволяет легко понять куда делось место на HDD или SSD.

Spybot Anti-Beacon [ сайт / видео ] Оценивает и отображает уровень приватности в вашей ОС, отключает телеметрию и шпионаж.

DWS [ сайт / видео ] Destroy Windows Spying – отрубает всю телеметрию Windows 10 одной кнопкой. Открыл – нажал – подождал – закрыл.

O&O ShutUp10 [ сайт / видео ] Твикер со множеством опций не только по отключению телеметрии, но и по настройкам приватности системы.

Диагностика, мониторинг, тестирование ПК

Программы для стресс тестов ПК после сборки или апгрейда, диагностики состояния жесткого диска, а также мониторинга параметров работающей системы.

Victoria HDD 5.28   [ сайт / видео / инфо ] Проверка диска на битые сектора. Работает под windows 7, 8, 10 x32 x64 ( victoria hdd for windows 7,8,10 ).

OCCT [ сайт  / видео ]  Лучшая программа для стресс теста ПК, позволяет мониторить напряжение, частоты и температуры. 

Furmark   [ сайт ] Небольшая утилита. Позволяет проводить стресс-тест видеокарт, создавая нагрузку, как в тяжелых играх.

CpuZ   [ сайт ] Очень подробная информация о процессоре и оперативной памяти. Есть встроенный тест и сравнение CPU между собой.

GpuZ   [ сайт ] Очень подробная информация о видеокарте и ее графическом процессоре. Поддерживает карты Nvidia и AMD.

HW INFO  [ сайт ] Отображает список оборудования, мониторит частоты, температуры, скорости вентиляторов и т.д.

HW Monitor  [ сайт ] Показания датчиков, мониторинг температур, скорости вентиляторов и т.д.

Open Hardware Monitor [ сайт ] мониторинг частот, температур, напряжений, скоростей вращения вентиляторов, загрузки. Можно вывести в трей.

Piriform Speccy [ сайт ] От создателей CCleaner – отображение компонентов системы и их температуры.

H2testw 1.4 rus [ сайт ] Программа для тестирования флешек на их реальную емкость, а также скорость чтения и записи. На русском.
